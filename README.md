### 🌐 [Live Demo Link](https://kpi-tracking-dashboard-ai-surveillance-web-app-8ft6chbrcc7s4xr.streamlit.app/)

## 📌 Executive Summary
This project demonstrates a real-time monitoring solution for a **B2B AI Video Analysis System**. As a Product Manager, I designed this dashboard to bridge the gap between raw model performance and business-level reliability. 

In a local surveillance environment, hardware constraints and environmental changes (lighting/weather) can cause **Model Drift**. This tool allows operations teams to monitor system health and act before a security failure occurs.

## 🚀 Key Features & KPIs
- **Inference Accuracy Tracking:** Monitors model reliability against defined SLA thresholds (85%).
- **P95 Latency Analysis:** Identifies bottlenecks in the edge-to-cloud data pipeline.
- **Fleet Uptime:** Provides a bird's-eye view of device connectivity across multiple locations.
- **Resource Allocation:** Visualizes GPU utilization on edge devices to optimize stream counts.

## 🛠️ Tech Stack
- **Backend:** Python
- **Interface:** Streamlit
- **Visuals:** Plotly (Interactive Charts)
- **Deployment:** Streamlit Community Cloud

## 📈 Why this matters for Product Management
Successfully deploying an AI product is only 50% of the job. The remaining 50% is ensuring the product continues to deliver value post-launch. This dashboard enables:
1. **Proactive Maintenance:** Catching accuracy dips before the customer complains.
2. **Data-Driven Roadmap:** Identifying latency issues to prioritize backend optimizations.
3. **Transparency:** Building trust with B2B stakeholders through visible performance metrics.
